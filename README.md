## Run Web
- Is exist dhttps as global package in your path?
- If it does not exist, type: pub global activate [dhttpd](https://pub.dev/packages/dhttpd)
- Plug your package into terminal to run this subject, type : dhttpd --path=build/web/ --host=127.0.0.1 --port=2010

## Build Web by Dart Compiler
- You can build web as your imagine as perfectly.
- Type : flutter build web --release
